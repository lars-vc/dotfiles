## Dependencies

### i3lock
i3lock supplies a lockscreen, with my [lock.sh](https://github.com/lars-vc/dotfiles/tree/root/private_dot_config/i3/i3scripts/executable_lock.sh) script there are a couple more dependencies:
* i3lock
* imagemagick
* scrot
